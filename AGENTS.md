## Agent skills

### Issue tracker

Rad se prati lokalno u Markdownu. `to-tickets` objavljuje odobreni plan u korijenski `tickets.md`, dok ostali engineering workflowi koriste `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Koriste se zadane oznake: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human` i `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Repo koristi single-context raspored s korijenskim `CONTEXT.md` i arhitektonskim odlukama u `docs/adr/`. See `docs/agents/domain.md`.
