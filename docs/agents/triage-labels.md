# Triage Labels

Engineering skillovi koriste pet kanonskih uloga. Ova tablica mapira te uloge na oznake lokalnog trackera.

| Uloga | Oznaka u trackeru | Značenje |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Održavatelj treba procijeniti zadatak |
| `needs-info` | `needs-info` | Čeka se dodatna informacija prijavitelja |
| `ready-for-agent` | `ready-for-agent` | Potpuno specificirano i spremno za samostalan rad agenta |
| `ready-for-human` | `ready-for-human` | Zahtijeva ljudsku implementaciju |
| `wontfix` | `wontfix` | Neće biti implementirano |

Kada skill spomene kanonsku ulogu, koristi odgovarajuću oznaku iz ove tablice.
